# Компонент товаров

## Отображение карточки

- Изображение
- Название
- Страна
- Ценой (с форматированием валют)

✅ Адаптивный дизайн
✅ Типизация пропсов

## Структура

```src/
├── components/
│   └── ProductCard.tsx  # Компонент карточки
├── mocks/
│   └── products.ts      # моковые данные
├── utils/
│   └── i18n.ts          # форматирование валют
└── App.tsx              # главный компонент
```

## Технологии

- **React 18**
- **TypeScript**
- **Material-UI (v5)**
- **Vite** (сборка)
